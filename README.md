# 超级骆狗工具箱 v3.0

Windows 一站式反流氓/系统维护工具，集成 9 大模块：扫描清理、反劫持、空间管理、安全审计、网络工具、装机助手、性能优化。

## 功能总览

### 1. 流氓软件扫描与清理（强力删除）
- 多盘扫描 / 目录 / 文件 / 注册表 / 服务 / 计划任务 / 启动项
- **四级强力删除**：直接删 → 杀占用进程 → takeown+icacls 提权 → MoveFileEx 重启删除
- 清理前自动**斩断守护链**：禁用服务 + 禁用任务 + taskkill 已知流氓进程
- 注册表 ACL 拒绝时 PowerShell 强夺所有权
- 大多数情况下无需进安全模式

### 2. 系统缓存清理
- 临时文件、预读取、浏览器缓存、Windows更新缓存、崩溃转储等

### 3. 反劫持检测
- **Hosts 文件** 检查 + 一键备份还原（自动刷新DNS）
- **浏览器快捷方式** 劫持检测 + 清除劫持参数
- **右键菜单** 流氓注册项扫描与删除
- **全部启动项总览**（注册表 Run + RunOnce + 启动文件夹），可疑项自动高亮

### 4. 空间清理增强
- **大文件查找** Top N（默认 ≥100MB Top30，跳过系统目录）
- **重复文件查找**（按文件大小+SHA1，显示可释放空间）

### 5. 安全审计套件
- **可疑进程扫描**：路径在 Temp/AppData/Downloads/回收站，附 Authenticode 签名状态
- **Defender 排除项审计**：列出路径/扩展名/进程白名单
- **未签名驱动检查**

### 6. 网络工具集
- **DNS / 代理设置检查**：识别可疑 ProxyEnable / AutoConfigURL 劫持
- **端口占用查询**：定位占用端口的 PID 与进程名
- **网络诊断一键化**：flushdns / release / renew / winsock reset / TCPIP reset / firewall reset

### 7. 系统信息
- 主机、CPU、内存、磁盘使用率彩色条形图

### 8. 装机助手（小白模式 · 新）
- **重置此电脑** —— 调系统自带 `systemreset.exe`，可保留个人文件
- **制作系统U盘指南** —— 自动检测U盘 + 微软/Rufus/Ventoy 三方案选择
- **进入高级启动** —— 一键重启进 WinRE（安全模式/PE/系统还原入口）
- **装机小白速查** —— 4 类场景图文教程 + 反流氓装机忠告

### 9. 性能优化（智能识别设备 · 新）
- **自动识别笔记本/台式机**（基于 `Win32_Battery`）
- **自动识别 SSD/HDD**（基于 `Get-PhysicalDisk`）
- **一键差异化优化**：
  - 台式机 → 高性能电源 + CPU 最小状态 100%
  - 笔记本 → 平衡模式（兼顾续航发热）
  - SSD → 关闭 SysMain (SuperFetch)；HDD → 保留
- **关闭透明 / 阴影**（窗口动画完全保留）
- **关闭遥测 / 广告 ID / 锁屏建议 / 推送**
- **电池健康报告**（仅笔记本，HTML 报告）
- **完整可逆**：原始设置自动备份到 `perf_backup.json`，一键还原

## 使用方法

### 方法一：直接运行Python脚本（需要Python环境）

```bash
python scanner.py
```

> 需要以管理员权限运行以获得完整扫描结果

### 方法二：打包为exe

1. 确保已安装Python 3.8+
2. 在Windows上双击运行 `build.bat`
3. 生成的exe在 `dist/360扫描器.exe`

或手动执行：
```bash
pip install pyinstaller
pyinstaller --onefile --name "360扫描器" --uac-admin scanner.py
```

### 运行要求

- Windows 7/10/11
- 管理员权限（推荐，用于完整注册表和服务扫描）

## 输出

- 控制台彩色输出扫描结果
- 自动生成 `scan_report_YYYYMMDD_HHMMSS.txt` 报告文件

## 注意事项

- 本工具支持**扫描 + 清理 + 系统优化**，所有清理/优化前都有二次确认
- 性能优化项可一键还原（备份在 `perf_backup.json`）
- 反劫持/装机助手中的"重启/重置"类操作需输入完整 `yes` 才会执行
- 建议以管理员身份运行以获得最完整的扫描和清理能力（首次启动会自动 UAC 提权）
- 隐藏文件夹会被特别标注 `[隐藏]`
