# 流氓软件扫描器 v2.0

全面扫描所有磁盘驱动器，查找隐藏的国产流氓/捆绑/垃圾软件残留。

## 功能

- **多盘扫描** - 自动检测所有固定磁盘并逐一扫描
- **目录扫描** - 查找已知安装目录（包括隐藏属性目录）
- **文件扫描** - 搜索流氓软件的关键可执行文件
- **注册表扫描** - 检测残留注册表项
- **服务扫描** - 查找可疑Windows服务
- **计划任务扫描** - 发现隐藏的定时任务
- **启动项扫描** - 检查开机自启项

## 使用方法

### 方法一：直接运行Python脚本（需要Python环境）

```bash
python scanner.py
```

> 需要以管理员权限运行以获得完整扫描结果

### 方法二：打包为exe

1. 确保已安装Python 3.8+
2. 在Windows上双击运行 `build.bat`
3. 生成的exe在 `dist/360扫描器.exe`

或手动执行：
```bash
pip install pyinstaller
pyinstaller --onefile --name "360扫描器" --uac-admin scanner.py
```

### 运行要求

- Windows 7/10/11
- 管理员权限（推荐，用于完整注册表和服务扫描）

## 输出

- 控制台彩色输出扫描结果
- 自动生成 `scan_report_YYYYMMDD_HHMMSS.txt` 报告文件

## 注意事项

- 本工具**仅扫描不删除**，不会修改任何系统文件
- 建议以管理员身份运行以获得最完整的扫描结果
- 隐藏文件夹会被特别标注 `[隐藏]`
